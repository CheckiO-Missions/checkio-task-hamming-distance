A Hamming-távolság – az két bináris egész szám közötti eltérő bitek számát jelenti ( bővebben a Hamming-távolságról a Wikipédiában olvashatunk).
Például: 

    117 = 0 1 1 1 0 1 0 1
     17 = 0 0 0 1 0 0 0 1
      H = 0+1+1+0+0+1+0+0 = 3

Adott két (N, M) pozitív decimális érték. Meg kell határozni e két érték binárisan ábrázolt alakja közötti Hamming-távolságot.

Alkalmazása:

E feladat alapját képezi a Hamming-kódnak és más lineáris hibajavító szoftvereknek. A Hamming-távolságot a rendszertanban is alkalmazzák a genetikai távolság meghatározására. Egy rácsstruktúrában (grid) (pl. a sakktáblán), a Hamming-távolság – az a bástya által az egyik mezőről a másikra tett minimális lépésszámot jelenti.

Előfeltételek:

  • 0 < n, m < 106
