Расстояние Хэмминга между двумя двоичными числами - это число позиций, в которых биты различаются (прочитать о расстоянии Хэмминга на Википедии).
Для примера: 

    117 = 0 1 1 1 0 1 0 1
     17 = 0 0 0 1 0 0 0 1
      H = 0+1+1+0+0+1+0+0 = 3
