Odległość Hamminga między dwiema binarnymi liczbami całkowitymi to liczba pozycji bitowych, które różnią się między sobą (read more about the Hamming distance on Wikipedia). Na przykład: 

    117 = 0 1 1 1 0 1 0 1
     17 = 0 0 0 1 0 0 0 1
      H = 0+1+1+0+0+1+0+0 = 3

Otrzymują Państwo dwie dodatnie liczby całkowite N, M w postaci dziesiętnej. Proszę obliczyć odległość Hamminga między tymi dwiema liczbami w postaci binarnej.

Sposób użycia:

Jest to podstawa kodu Hamminga i innych liniowych programów korygujących błędy. Odległość Hamminga jest używana w systematyce jako miara odległości genetycznej. Na siatce (np. szachownicy) odległość Hamminga to minimalna liczba ruchów potrzebnych wieży do przejścia z jednej komórki do drugiej.
