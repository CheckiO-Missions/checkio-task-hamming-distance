二つの2進数の間のハミング距離は異なるビットの数である。 (ウィキペディア「ハミング距離」). 例えば: 

    117 = 0 1 1 1 0 1 0 1
     17 = 0 0 0 1 0 0 0 1
      H = 0+1+1+0+0+1+0+0 = 3

あなたには二つの正数(N, M)が10進数で与えられる。 各々が2進数の場合のハミング距離を計算すること。

これはハミング符号や他の線形エラー訂正プログラムの基礎である。 ハミング距離は分類学において遺伝的距離の計測に使用される。 グリッド上(すなわちチェス盤) では ハミング距離はルークがあるマスへ移動する際の最小の移動回数となる。
